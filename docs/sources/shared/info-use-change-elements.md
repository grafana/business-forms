{{< admonition type="note" >}}
Use this model to set elements if you use the [`context.panel.onChangeElements([])`](https://grafana.com/docs/plugins/volkovlabs-form-panel/<PLUGINS_VERSION>/custom-code/context-parameters/#panelonchangeelementsoptions) method.

Dynamic element creation and manipulation through the panel fetches the complete element model and sets it through a method. The elements aren't saved in the panel options.
{{< /admonition >}}
