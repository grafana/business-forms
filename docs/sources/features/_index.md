# Features

The Business Forms panel provides an extensive set of features.

| Section                                  | Description                                                                       |
| ---------------------------------------- | --------------------------------------------------------------------------------- |
| [Control panel](https://grafana.com/docs/plugins/volkovlabs-form-panel/<PLUGINS_VERSION>/features/control-panel/)              | Create control panels with multiple sections and the same identifier.             |
| [Custom requests](https://grafana.com/docs/plugins/volkovlabs-form-panel/<PLUGINS_VERSION>/features/custom-requests/)            | Create initial and update requests with custom code.                              |
| [Dynamic forms](https://grafana.com/docs/plugins/volkovlabs-form-panel/<PLUGINS_VERSION>/features/dynamic-forms/)              | Update form element values and options from any data source.                      |
| [Variables](https://grafana.com/docs/plugins/volkovlabs-form-panel/<PLUGINS_VERSION>/features/variables/)                | Replace dashboard and global variables.                                           |
| [Load indication](https://grafana.com/docs/plugins/volkovlabs-form-panel/<PLUGINS_VERSION>/features/load/)               | View how Business Forms indicates loading progress.                               |
| [Confirmation window](https://grafana.com/docs/plugins/volkovlabs-form-panel/<PLUGINS_VERSION>/features/confirm-window/) | Configure the confirmation window that displays before submitting form updates.   |
